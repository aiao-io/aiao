# @aiao/lazy-component

按需加载 任意 `module` 的 `component`. https://github.com/angular/angular/issues/14324 的解决方案

> angular 9 有新的方式还没研究. 此方法暂时适用 angular 8

## 场景

弹框弹出任意组件, 在后台页面操作数据时使用非常频繁

## 安装

```console
yarn add @aiao/lazy-component
```
