# @aiao/lazy-component

### 按需加载 任意 `module` 的 `component`.

angular 9 有新的方式. 此方法适用 angular 8 及以前版本

## 以下错误的解决方案

https://github.com/angular/angular/issues/14324#issuecomment-559383535

## 使用场景

弹框弹出任意组件, 在后台页面操作数据时使用非常频繁

## Install

```console
yarn add @aiao/lazy-component
```
