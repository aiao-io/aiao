# @aiao/stencil-toolkit

[![npm][shields-stencil-toolkit]][npm-stencil-toolkit]

`stencil` 的工具集 , 支持 `nx`

## 场景

动态显示内容

## 安装

```console
yarn add @aiao/stencil-toolkit --dev
```

## 使用

```
ng generate @aiao/stencil-toolkit:library you-library-name
```
