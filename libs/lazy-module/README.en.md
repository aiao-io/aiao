# @aiao/lazy-module

[![npm](https://img.shields.io/npm/v/@aiao/lazy-module?style=flat-square)](https://www.npmjs.com/@aiao/lazy-module)

lazy load `angular module`

## Use Case

不用等到路由触发后才加载相应模块，可以由代码控制主动预加载，提高用户体验

## Install

```console
yarn add @aiao/lazy-module
```
