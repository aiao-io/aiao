# @aiao/lazy-module

[![npm][shields-lazy-module]][npm-lazy-module]

按需加载 `angular module`

## 场景

不用等到路由触发后才加载相应模块，可以由代码控制主动预加载，提高用户体验

## 安装

```console
yarn add @aiao/lazy-module
```
