# @aiao/lazy-module

[![npm][shields-lazy-module]][npm-lazy-module]

按需加载 `angular module`

## 场景

动态加载模块

## 安装

### Angular 8

```console
yarn add @aiao/lazy-module@^1.3
```

### Angular 9+

```console
yarn add @aiao/lazy-module
```
